## Ciclos anidados
## Ciclos anidados: Ejemplo en C++ ```cpp #include <iostream> using namespace std; void imprimir(int a, int b) { for(int i = a; i <= b; i++) { for(int j = 1; j <= 12; j++) { cout << j << "/" << i << endl; } } } int main() { int a = 0; int b = 0; cout << "Ingrese el primer a単o (yy): \t"; cin >> a; cout << "Ingrese el segundo a単o (yy): \t"; cin >> b; imprimir(a, b); return 0; } ```
## Ciclos anidados: Ejemplo en Python ```python def imprimir(a, b): for i in range(a, b + 1): for j in range(1, 13): print(str(j) + "/" + str(i)) a = int(input("Ingrese el primer a単o (yy): \t")) b = int(input("Ingrese el segundo a単o (yy): \t")) imprimir(a, b) ```
